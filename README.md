# burp-labs 靶场实验室

一个专为学习Burp Suite安全测试工具设计的综合性靶场项目，包含多个难度递增的关卡，覆盖了Web安全测试中的各种常见场景和技术。

## 📋 项目概述

本项目旨在通过实践的方式帮助安全爱好者、渗透测试人员和开发人员掌握Burp Suite的使用技巧。每个关卡都模拟了真实世界中的安全场景，从基础的爆破攻击到复杂的加密和频率限制绕过。

### 🎯 学习目标
- 掌握Burp Suite Intruder模块的各种攻击模式
- 理解不同编码和加密方式下的爆破技巧
- 学习处理各种响应过滤和限制机制
- 培养Web应用安全测试的实战能力

## 🚀 快速开始

### 环境要求
- PHP 7.0+
- Web服务器 (Apache/Nginx)
- Burp Suite Professional
- 现代浏览器

### 安装步骤
1. 克隆项目到Web服务器目录：
```bash
git clone https://github.com/g0dxing/burp-labs.git
cd burp-labs
```

2. 配置Web服务器，确保PHP支持开启

3. 启动Burp Suite并配置浏览器代理

4. 从Level 1开始逐个挑战

### 使用指南
1. **配置代理**：设置浏览器代理为127.0.0.1:8080
2. **捕获流量**：使用Burp Proxy拦截请求
3. **发送到Intruder**：右键点击请求 → Send to Intruder
4. **配置攻击**：根据关卡说明设置攻击参数和payloads
5. **分析结果**：使用过滤器识别成功响应
